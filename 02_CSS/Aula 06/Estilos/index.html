No CSS devemos ter o seletor inicialmente seguido de {} por exemplo p {}. Existem 3 fotos de aplicar o CSS.


inline - Aplicado no próprio elemento, como na linha 20, colocamos o style dentro da tag do elemento.
interno - Abre-se uma tag chamada style no meta, conforme linha 8 deste código.

externo - Criaremos um arquivo .css e o linkaremos no meta atraves da tag link

A prioridade de aplicação será a mais perto do elemento em caso de existirem as 3 utilizações. Dessa forma primeiro a inline, depois a interna e por fim a externa.

No CSS os comentários são feitos no seguinte formato /*Comentário*/
