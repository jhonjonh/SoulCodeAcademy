Um pouco sobre nós

Desenvolvimento

Nossa equipe tem como missão o desenvolvimento de excelentes páginas web. Nesse projeto focamos no desenvolvimento de uma página com conversor cambial e de página com calculadora comum e de calculadora científica. O objetivo fpi a consolidação do conhecimento de HTML, CSS e JavaScript obtido até aqui. Foco na integração do conhecimento individual sendo compartilhado a todos os integrantes deste grupo.
